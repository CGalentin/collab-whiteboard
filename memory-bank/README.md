# Memory bank (project context)

Structured notes so you (and Cursor) can reload **what matters** without rereading every doc.

## How to use in Cursor

- **@ mention** these files in Chat or Composer when starting a session or a big task (e.g. `@memory-bank/activeContext.md` + `@memory-bank/techContext.md`).
- **Update** `activeContext.md` and `progress.md` whenever you change focus or finish a milestone—keep them short.
- **Source of truth** for full detail: [PRESEARCH_AND_TRACKING.md](../PRESEARCH_AND_TRACKING.md), [BUILD_ROADMAP.md](../BUILD_ROADMAP.md), [docs/GAUNTLET_FULLSTACK_BEST_PRACTICES.md](../docs/GAUNTLET_FULLSTACK_BEST_PRACTICES.md).
